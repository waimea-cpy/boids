# Boids

A JavaScript implementation of the classic Boids flocking algorithm.

## About

Boids is an artificial life simulation that demonstrates emergent flocking behaviour through three simple steering rules: separation, alignment, and cohesion.

Learn more: [Boids on Wikipedia](https://en.wikipedia.org/wiki/Boids)

## Usage

Open `index.html` in a web browser to see the simulation in action.
